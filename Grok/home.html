Quick Git Workflow

  1. git checkout -b feature/{name}
  2. git add -A
  3. git commit -m "Comment"
  4. git checkout main
  5. git merge feature/{name}
  6. git branch -d feature/{name}
  7. git push -v origin main

More Git commands


Start workflow
git branch -d feature/oldname(cleanup: remove last local feature branch [feature/oldFeature])
git push origin -delete feature/oldname (cleanup: remove last remote feature branch [origin feature/oldFeature])

Refresh Branches
git checkout main(goto local main branch root folder)
git pull origin main (refresh local current main)
git checkout develop (goto develop branch)
git pull origin develop(refresh local develop branch)

Begin Coding
git checkout -b feature/name (create local feature branch from develop)

Coding done, PR Checklist done
git add -A (stage changed files)
git commit -m "Feature comment" (commit staged changes to feature branch)

Ready for PR
git checkout develop (goto local main branch root folder)
git merge feature/name (pull feature changes into local develop branch)
git push origin feature/name (push local feature branch to remote server for Pull Request)

Address PR comments
git checkout develop (Reviewers could have changed remote feature branch during review)
git pull origin/feature/name check for changes, pull as needed; verify this syntax
git checkout feature/name goto local feature branch

Address Comments, Code Again
git add -A (stage all changes to current branch)
git commit -m "PR comments"(commit staged changes to local feature branch)
git checkout develop(goto local root develop branch)
git merge feature/name (pull feature changes into local develop branch)
git push origin feature/name(push local feature branch to remote)
